# Linux shortcut


### to minimize
~~~
alt + space + N
~~~


### to lock
~~~
windows + l
~~~

### to open terminal
~~~
ctrl + alt + t
~~~

### to open a new tab inside terminal
~~~
ctrl + shift + t
~~~

### to open a new window in terminal
~~~
ctrl + shift + w
~~~

### to close a tab inside terminal
~~~
ctrl + shift + w 
~~~
